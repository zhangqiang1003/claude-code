# 模型选型指南

根据您的具体业务场景和需求，推荐选择以下模型以获得最佳效果：

| 应用场景                                                       | 推荐模型                   | 推荐理由                                                                                                                            |
| :------------------------------------------------------------- | :------------------------- | :---------------------------------------------------------------------------------------------------------------------------------- |
| **声音设计**(品牌形象、专属声音、扩展系统音色等)_基于文本描述_ | `qwen3-tts-vd-2026-01-26`  | **支持声音设计 (Voice Design)**。无需提供音频样本，仅需通过文本描述即可创建定制化音色。适合从零开始设计独特的品牌专属声音。         |
| **声音复刻**(品牌形象、专属声音、扩展系统音色等)_基于音频样本_ | `qwen3-tts-vc-2026-01-22`  | **支持声音复刻 (Voice Cloning)**。基于真实音频样本快速复刻音色，能够打造高度拟人化的品牌声纹，确保音色的高度还原与一致性。          |
| **情感化内容生产**(有声书、广播剧、游戏/动画配音)              | `qwen3-tts-instruct-flash` | **支持指令控制 (Instruct)**。可通过自然语言描述精确控制音调、语速、情感状态及角色性格。非常适合需要丰富表现力和复杂角色塑造的场景。 |
| **移动端导航 / 通知播报**                                      | `qwen3-tts-flash`          | **高性价比 & 低延迟**。按字符计费，模式简单透明。专为短文本、高频调用场景优化，响应速度快。                                         |
| **在线教育课件配音**                                           | `qwen3-tts-flash`          | **多语种 & 方言支持**。能够灵活应对不同地域的教学需求，支持多种语言和方言的流畅播报。                                               |
| **有声读物批量生产**                                           | `qwen3-tts-flash`          | **成本可控 & 音色丰富**。在保持低成本的同时，提供丰富的音色选择，能够满足长篇内容生产的多样化表现力需求。                           |

---

### 💡 选型建议总结

- **需要全新定制声音但无录音？** $\rightarrow$ 选 **声音设计模型** (`...vd-...`)
- **已有录音需要克隆声音？** $\rightarrow$ 选 **声音复刻模型** (`...vc-...`)
- **需要精细控制情感和表演？** $\rightarrow$ 选 **指令控制模型** (`...instruct-...`)
- **追求极致性价比、短文本或大规模生产？** $\rightarrow$ 选 **Flash 模型** (`...flash`)

### 3. 各模型支持的语种能力

#### **Qwen3-TTS-Flash / Realtime**

- **核心能力**：支持 **10+ 种主流语言** 及 **8+ 种中国方言** 的混合输入与输出。
- **支持语种列表**：
  - **中文** (普通话及各方言)
  - **英文** (English)
  - **日语** (Japanese)
  - **韩语** (Korean)
  - **法语** (French)
  - **西班牙语** (Spanish)
  - **葡萄牙语** (Portuguese)
  - **德语** (German)
  - **俄语** (Russian)
  - **意大利语** (Italian)
  - **土耳其语**、**阿拉伯语**、**印地语**、**泰语**、**越南语**等 (具体视版本更新而定，最新版支持范围极广)。
- **混合能力**：支持**中英混读**、**多语种段落自动切换**，无需手动指定语言参数，模型会根据文本自动识别并调整发音。

#### **CosyVoice-v3-Flash**

- **支持语种**：主要集中在 **中文** 和 **英文**，对双语混合支持极佳。
- **特点**：在情感爆发力（如大笑、哭泣、愤怒）方面表现优于普通TTS模型。

#### **Sambert**

- **支持语种**：中文、英文、日语、韩语等主要语种。
- **特点**：稳定性高，但在复杂多语种混合场景下不如Qwen3灵活。

### 4. 关键特性与使用说明

1.  **指令控制 (Instruct)**：
    - Qwen3-TTS-Flash 支持通过自然语言指令控制语音表现。
    - **用法**：在调用API时，除了传入文本，还可以传入指令（如：“请用悲伤的语气朗读这段话”、“语速放慢，像讲故事一样”）。
    - **模型选择**：需使用 `qwen3-tts-instruct-flash` 或在参数中开启指令模式。

2.  **声音复刻 (Voice Clone)**：
    - **功能**：上传10秒~1分钟的音频，即可克隆出专属音色。
    - **流程**：先调用 `CreateVoice` (声音复刻) 接口获取自定义音色ID，再将该ID用于 `SpeechSynthesis` 接口。
    - **支持模型**：Qwen3-TTS-Flash 和 CosyVoice 均强力支持。

3.  **声音设计 (Voice Design)**：
    - **功能**：通过文字描述生成不存在的音色（例如：“一个带有烟嗓的中年男性，说话略带慵懒”）。
    - **模型**：`qwen-voice-design`。

4.  **音频格式与采样率**：
    - **格式**：支持 PCM, WAV, MP3, OGG/OPUS。
    - **采样率**：支持 8kHz, 16kHz, 22.05kHz, 24kHz, 44.1kHz, **48kHz** (高保真)。

### 5. 如何获取完整音色列表？

由于音色库会随版本动态更新（目前官方宣称为49+种），最准确的实时列表建议通过以下方式获取：

1.  **阿里云百炼控制台**：登录 [阿里云百炼](https://bailian.console.aliyun.com/) -> 模型广场 -> 语音合成 -> 试听页面，可直接试听所有可用音色。
2.  **API 查询**：部分SDK或API接口支持查询可用音色列表（ListVoices），但通常直接使用文档中列出的常用ID（如 `longwan`, `Cherry` 等）最为稳妥。

**总结建议**：

- 若追求**最高拟人度和多语种能力**，首选 **`qwen3-tts-flash`**，音色推荐 **`longwan`** (通用女声) 或 **`longcheng`** (通用男声)。
- 若需要**实时对话**，使用 **`qwen3-tts-flash-realtime`**。
- 若需要**极度情感化**或**声音复刻**，可尝试 **`cosyvoice-v3-flash`**。

# 文生视频素材编辑功能实现计划

## Context

在文生视频 pipeline 中，DraftPopulateStep 完成素材准备后，需要增加一个用户手动编辑步骤，让用户可以在生成剪映草稿前审核和修改素材数据（视频、音频、字幕、背景音乐）。

**需求要点**：
- 新对话框展示素材数据
- 用户可编辑字幕、视频、背景音乐
- 用户可跳过编辑使用原始数据
- 用户取消时保存数据等待续传
- 数据保存到数据库支持断点续传

     constructor(window: BrowserWindow, db: DB) {
       super();

       this.db = db;

2. 实现 `execute` 方法：
   - 从 context 获取 DraftPopulateStep 准备的素材数据
   - 更新数据库审核状态为 1（审核中）
   - 发送 `draft-review:request` 事件到渲染进程
   - 等待用户响应（Promise 等待）
   - 处理三种响应结果（confirm/skip/cancel）
   - 返回 StepResult

3. 实现等待机制：
   ```typescript
   private pendingResolves: Map<number, (value: UserEditResult) => void> = new Map();

   private waitForUserResponse(taskId: number): Promise<UserEditResult> {
     return new Promise((resolve) => {
       this.pendingResolves.set(taskId, resolve);
     });
   }

   // IPC 调用此方法恢复执行
   handleUserResponse(taskId: number, result: UserEditResult): void {
     const resolve = this.pendingResolves.get(taskId);
     if (resolve) {
       resolve(result);
       this.pendingResolves.delete(taskId);
     }
   }
   ```

4. 实现 `canSkip` 方法（断点续传）：
   ```typescript
   canSkip(context: TextToVideoContext, taskRecord: any): boolean {
     // 状态为 2（已确认）或 3（已跳过）时跳过
     return taskRecord?.draft_review_status === 2 ||
            taskRecord?.draft_review_status === 3;
   }
   ```

### Task 3.1: 添加 IPC 处理器

**文件**: `frontend/src/main/ipc_handle/index.ts`

**步骤**:
1. 创建 `registerDraftReviewHandlers` 方法：
   ```typescript
   private registerDraftReviewHandlers(): void {
     // 提交编辑结果
     ipcMain.handle('draft-review:submit', async (_, taskId: number, result: UserEditResult) => {
       // 1. 保存编辑数据到数据库
       if (result.action === 'confirm' && result.editedMaterials) {
         this.db.updateTextToVideoTaskEditedMaterials(
           taskId,
           JSON.stringify(result.editedMaterials.videoInfos),
           JSON.stringify(result.editedMaterials.audioInfos),
           JSON.stringify(result.editedMaterials.textInfos),
           JSON.stringify(result.editedMaterials.bgMusicConfig)
         );
         this.db.updateTextToVideoTaskReviewStatus(taskId, 2);
       } else if (result.action === 'skip') {
         this.db.updateTextToVideoTaskReviewStatus(taskId, 3);
       } else if (result.action === 'cancel') {
         this.db.updateTextToVideoTaskReviewStatus(taskId, 4);
       }

       // 2. 恢复 pipeline 执行
       const step = this.pendingReviewSteps.get(taskId);
       if (step) {
         step.handleUserResponse(taskId, result);
         this.pendingReviewSteps.delete(taskId);
       }

       return { success: true };
     });

     // 获取审核数据（断点续传用）
     ipcMain.handle('draft-review:get-data', async (_, taskId: number) => {
       const task = this.db.getTextToVideoTask(taskId);
       return {
         videoInfos: task.edited_video_infos ? JSON.parse(task.edited_video_infos) : null,
         audioInfos: task.edited_audio_infos ? JSON.parse(task.edited_audio_infos) : null,
         textInfos: task.edited_text_infos ? JSON.parse(task.edited_text_infos) : null,
         bgMusicConfig: task.edited_bg_music_config ? JSON.parse(task.edited_bg_music_config) : null,
         reviewStatus: task.draft_review_status,
       };
     });
   }
   ```

2. 在 `register` 方法中调用：
   ```typescript
   register(): void {
     // ... 其他注册
     this.registerDraftReviewHandlers();
   }
   ```

**验收标准**:

## Execution Flow

```
DraftPopulateStep 完成
       ↓
DraftReviewStep 开始
       ↓
发送 'draft-review:request' 到渲染进程
       ↓
【PAUSE】等待用户响应
       ↓
渲染进程打开 DraftReviewDialog
       ↓
用户操作：
  ├─ 跳过 → 使用原始数据继续
  ├─ 取消 → 保存状态，停止执行
  └─ 确认 → 保存编辑数据，使用新数据继续
       ↓
DraftGenerateStep 执行
```

---

## Resume Support

**断点续传逻辑**：
1. 检查 `draft_review_status` 字段
2. 状态为 2（已确认）或 3（已跳过）时，跳过 DraftReviewStep
3. 状态为 1（审核中）时，重新打开审核对话框
4. 从数据库加载已保存的编辑数据

## Data Types

```typescript
// 用户编辑结果
interface UserEditResult {
  action: 'confirm' | 'skip' | 'cancel';
  editedMaterials?: {
    videoInfos: VideoInfoRequest[];
    audioInfos: AudioInfoRequest[];
    textInfos: TextInfoRequest[];
    bgMusicConfig: BgMusicConfig[];
  };
}

// 审核请求数据
interface DraftReviewRequest {
  taskId: number;
  videoInfos: VideoInfoRequest[];
  audioInfos: AudioInfoRequest[];
  textInfos: TextInfoRequest[];
  bgMusicConfig: BgMusicConfig[];
}

// 审核状态枚举
enum DraftReviewStatus {
  NOT_REVIEWED = 0,  // 未审核
  REVIEWING = 1,     // 审核中
  CONFIRMED = 2,     // 已确认
  SKIPPED = 3,       // 已跳过
  CANCELLED = 4,     // 已取消
}
```

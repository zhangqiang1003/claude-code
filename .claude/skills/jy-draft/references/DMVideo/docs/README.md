## 🔧 核心概念

### 时间单位
- 所有时间单位均为**微秒**（μs）
- `1 秒 = 1,000,000 微秒`

### 索引规则
- 数组索引从 **1** 开始（segment_index 等）

### 颜色格式
- RGB 数组：`[R,G,B]`（0-255）
- HEX 格式：`#RRGGBB`

### 透明度范围
- 取值范围：`0.0 ~ 1.0`
- 0.0 = 完全透明，1.0 = 完全不透明

## 📋 API 概览

| 模块 | 主要功能 |
|------|----------|
| 时间线 | 生成时间线分段、根据音频时长生成分段 |
| 音频 | 创建/修改音频片段、添加音效、淡入淡出 |
| 视频 | 创建/修改视频片段、特效、滤镜、转场、蒙版 |
| 文本 | 创建/修改文本片段、样式设置、动画效果 |
| 贴纸 | 创建/修改贴纸、位置变换 |
| 关键帧 | 位置、缩放、旋转、透明度、音量动画 |
| 草稿 | 创建草稿、添加素材、保存导出 |

---

## ⚠️ 注意事项

1. 所有时间单位均为**微秒**（μs）
2. 数组索引从 **1** 开始（部分 API）
3. 颜色格式：RGB 数组 `[R,G,B]` 或 HEX `#RRGGBB`
4. 透明度范围：`0.0 ~ 1.0`
5. 主视频轨道片段必须从 **0s** 开始
6. 转场应添加在**前面的**片段上
7. 背景填充仅对**底层视频轨道**上的片段生效

## 🚀 快速开始

### 创建一个简单视频草稿

```
1. create_draft(width=1920, height=1080)
2. add_videos(video_infos)      # 添加视频素材
3. add_audios(audio_infos)      # 添加背景音乐
4. add_texts(text_infos)        # 添加字幕
5. save_draft()                 # 保存获取链接
```

### 添加特效和动画

```
1. generate_video_effect()      # 生成视频特效 ID
2. add_video_effects(effect_ids) # 添加到草稿
3. generate_keyframe()          # 生成关键帧 ID
4. add_keyframes(keyframe_ids)  # 添加到草稿
```

## Style

- Learns best with: 深度源码分析 + 动手实践（04-multi-agent.ts 改造）+ 对比记忆（Claude Code vs demo 实现）

## Patterns

- 初期混淆：Compaction 的 5 级触发 vs 4 层操作（名字相似但维度不同）→ 通过对比表格澄清

- 习惯性地问"底层是不是同一套"：需要明确指出共享的底层实现
- Hook 执行管道：初期认为串链执行 → 源码验证后更正为并行执行
